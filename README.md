## 사용 방법

### 1. 서버

  1. 필요한 패키지 설치

  ```bash
  pip install -r requirements.txt
  ```

  2. 서버 시작

  ```bash
  uvicorn main:app --reload --host 0.0.0.0 --port 8000
  ```

### 2. 클라이언트

- 웹사이트 접속
